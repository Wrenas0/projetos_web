Listas não ordenadas

A tag ul (unordered list - lista não ordenada) representa uma lista de itens sem ordem rígida, por exemplo ingredientes em uma receita.

A tag li (list items - itens de lista) representa um item que faz parte de uma lista.

Exemplo de lista

  • primeiro item
  • segundo item
  • terceiro item
    • subitem

Lista ordenada

A tag ol (ordered list - lista ordenada) representa uma lista de itens numerados sequencialmente.

  1. primeiro item
  2. segundo item

Atributos

reversed (ordem reversa)
Exibe os itens da lista ordenada em ordem revesa.

  1. item mais importante
  2. item menos importante

start (inicio)
especifica o valor inicial para numeração da lista.

  1. item 5
  2. item 4

type (tipo)
indica o tipo de numeração:
a- indica letras minusculas:
A- indica letras maiusculas
i- indica algarismos romanos minusculos
I- indica algarismos romanos maiusculas

  1. primeiro item
  2. segundo item
